# Changelog

## 0.0.2

- Switched Safari to classic overworld generation (no reset/structure).
- Disabled player damage and block breaking/placing in Safari.
- Restricted in‑Safari item use to Safari Balls only.
- Added `/safari buy time <minutes>` pricing and 30‑minute default sessions.
- Added action bar warnings (red under 5 minutes + final 10s tick sound).
- Added Safari Guide NPC with ticket items (+5/+15/+30 min) and spawn egg.
- Added shop UI balance display and 16/32/64 Safari Ball bundles.
- Added Safari biome + custom Cobblemon spawn pools.
- Added session resume on reconnect and safari NPC auto-spawn at (0,0)+10.
- Tickets now have a 1s use cooldown and entry warning title.

## 0.0.1

- Safari dimension with custom seed‑based generator and daily resets.
- Portal frame + portal block with Nether‑style frame detection.
- Custom Safari Ball item with Cobblemon throw physics and sounds.
- Spawn structure placement at (0,0) in the sky.
- World‑specific config and economy integration.
- Player send‑out blocked inside Safari.
